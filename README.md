## Running `Mini-Spidey`
### Running `online`
- Go to https://rededge967.github.io/mini-spidey/
### Running `offline`
- Download `source.code.zip` from [here](https://github.com/RedEdge967/mini-spidey/releases)
- Extract the folder
- Run `index.html` file in the browser
